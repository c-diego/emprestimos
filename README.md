## Passo a Passo [Dev]

```git clone https://github.com/c-diego/emprestimos.git```  
```cd emprestimos```  
```cp .env.example .env```  
```npm i```  
```composer i```  
```php artisan key:generate```  
```php artisan migrate --seed```  
```npm run dev```  
```php artisan serve```
